# Mir Commander Chemistry Plugins

This is a set of advanced tools for structure manipulation and visualization.

## [Cartesian Editor](cartesian-editor/README.md)

A simple editor for `mircmd:chemistry:atomic_coordinates` objects.

## [Molecular Visualiszer](molecular-visualizer/README.md)

A molecular visualizer.

## [Files Importer](files-importer/README.md)

A set of importers for certain file formats.

## [Icons](object-icons/README.md)

A set of icons for objects with certain data.
