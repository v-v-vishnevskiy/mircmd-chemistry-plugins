# Files Importer

Can import molecular data from a wide variety of computational chemistry and molecular modeling file formats.

## Supported formats

- XYZ
- UNEX
- MDL Mol V2000
- Cfour
- Gaussian cube

## Output objects

### `mircmd:chemistry:molecule`

### `mircmd:chemistry:atomic_coordinates`

### `mircmd:chemistry:atomic_coordinates_group`

### `mircmd:chemistry:unex`

### `mircmd:chemistry:volume_cube`
